# OIBSIP - OASIS INFOBYTE Internship Program

## 📚 Project Overview

This repository contains data science projects completed during the **OASIS INFOBYTE Internship Program**. The internship is a 1-month learning opportunity that covers various aspects of data science, machine learning, and data analysis.

## 📁 Repository Structure

The repository is organized into two main task folders, each representing a separate data science project:

### **Task 1: Iris Flower Classification**
Directory: `Task 1 - Iris Flower Classification/`

**Objective**: Classify iris flowers into three species based on physical measurements using machine learning.

**Key Components**:
- Dataset: Iris.csv (150 flower samples)
- Models: Random Forest, Logistic Regression, SVM
- Evaluation Metrics: Accuracy, Precision, Recall, F1-Score
- Visualizations: Feature distributions, correlation matrix, confusion matrix

**Technologies**: Python, Pandas, Scikit-learn, Matplotlib, Seaborn

[More Details →](./Task%201%20-%20Iris%20Flower%20Classification/README.md)

---

### **Task 2: Unemployment Rate Analysis in India**
Directory: `Task 2 - Unemployment Analysis/`

**Objective**: Analyze unemployment trends in India from May 2019 to June 2020, with focus on COVID-19 pandemic impact.

**Key Components**:
- Dataset: Unemployment in India.csv (comprehensive regional data)
- Analysis: Pre-COVID vs COVID period comparison
- Regional Analysis: State-wise and urban/rural breakdowns
- Visualizations: 7 comprehensive charts and heatmaps
- Insights: Regional disparities, labour participation trends

**Technologies**: Python, Pandas, NumPy, Matplotlib, Seaborn

[More Details →](./Task%202%20-%20Unemployment%20Analysis/README.md)

---
